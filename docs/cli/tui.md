# `skynet tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
skynet tui
skynet tui --url ws://127.0.0.1:18789 --token <token>
skynet tui --session main --deliver
```
